## ✅ 如何使用

1. 將所有檔案上傳到你的 GitHub 倉庫（包含 HTML 和 audio 資料夾）
2. 前往 GitHub 倉庫的 Settings → Pages
3. 將 GitHub Pages 設定為使用 `main` 分支和根目錄 `/`，儲存
4. 開啟你的學習網站網址：  
   `https://你的帳號.github.io/你的倉庫名稱/`
