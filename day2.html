📝 Quiz

  1. What is a canvas?
    1. A kind of paint
    2. A type of brush
    3. ✔ A surface used for painting
    4. A drawing tool

    Answer: C — A canvas is a surface used for painting.

  2. What is the purpose of a sketch?
    1. To frame the artwork
    2. To fill in colors
    3. ✔ To create a rough outline
    4. To clean the brushes

    Answer: C — A sketch is a rough outline made before painting.

  3. Which item is used to support a canvas?
    1. Palette
    2. Brush
    3. ✔ Easel
    4. Table

    Answer: C — An easel supports the canvas.
